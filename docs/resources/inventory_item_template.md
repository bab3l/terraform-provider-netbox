# netbox_inventory_item_template (Resource)

Manages an inventory item template in NetBox. Inventory item templates define inventory items for device types.

## Example Usage

```terraform
resource "netbox_manufacturer" "test" {
  name = "Test Manufacturer"
  slug = "test-manufacturer"
}

resource "netbox_device_type" "test" {
  model        = "Test Model"
  slug         = "test-model"
  manufacturer = netbox_manufacturer.test.name
  u_height     = 1
}

resource "netbox_inventory_item_template" "test" {
  name        = "Inventory Item Template"
  device_type = netbox_device_type.test.model
}
```

## Schema

### Required

- `device_type` (String) The device type this inventory item template belongs to (ID or model name).
- `name` (String) The name of the inventory item template. {module} is accepted as a substitution for the module bay position when attached to a module type.

### Optional

- `component_id` (String) The ID of the component this inventory item represents.
- `component_type` (String) The type of component this inventory item represents (e.g., `dcim.interface`).
- `description` (String) Description of the inventory item template.
- `label` (String) Physical label of the inventory item template.
- `manufacturer` (String) The manufacturer of the inventory item (ID or slug).
- `parent` (String) Parent inventory item template (ID).
- `part_id` (String) Manufacturer-assigned part identifier.
- `role` (String) The inventory item role (ID or slug).

### Read-Only

- `id` (String) The unique numeric ID of the inventory item template.

## Import

Import is supported using the following syntax:

```shell
# Inventory Item Templates can be imported by ID
terraform import netbox_inventory_item_template.test 123
```
