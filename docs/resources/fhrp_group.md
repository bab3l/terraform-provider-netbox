# netbox_fhrp_group (Resource)

Manages an FHRP (First Hop Redundancy Protocol) group in Netbox. FHRP groups represent virtual IP configurations for protocols like VRRP, HSRP, CARP, GLBP, and others that provide gateway redundancy.

## Example Usage

```terraform
resource "netbox_fhrp_group" "test" {
  protocol  = "vrrp"
  group_id  = 10
  auth_type = "plaintext"
  auth_key  = "secret"
}
```
