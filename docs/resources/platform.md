# netbox_platform (Resource)

Manages a platform in Netbox. Platforms represent the software running on a device, such as an operating system or firmware version.

## Example Usage

```terraform
resource "netbox_manufacturer" "test" {
  name = "Cisco"
  slug = "cisco"
}

resource "netbox_platform" "test" {
  name            = "Cisco IOS"
  slug            = "cisco-ios"
  manufacturer_id = netbox_manufacturer.test.id
}
```

## Schema

### Required

- `name` (String) Name of the platform.
- `slug` (String) URL-friendly identifier for the platform. Must be unique and contain only lowercase letters, numbers, hyphens, and underscores.

### Optional

- `description` (String) Description of the platform.
- `manufacturer` (String) Reference to the manufacturer (ID or slug).

### Read-Only

- `id` (String) Unique identifier for the platform (assigned by Netbox).
