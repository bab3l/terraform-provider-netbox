# netbox_config_template (Resource)

Manages a config template in NetBox. Config templates are Jinja2 templates used to render configuration for devices.

## Example Usage

```terraform
resource "netbox_config_template" "test" {
  name          = "Test Config Template"
  template_code = "hostname {{ device.name }}"
  environment_params = jsonencode({
    "foo" : "bar"
  })
}
```
