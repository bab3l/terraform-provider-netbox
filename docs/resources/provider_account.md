# netbox_provider_account (Resource)

Manages a provider account in Netbox. Provider accounts represent accounts with circuit providers.

## Example Usage

```terraform
resource "netbox_provider" "test" {
  name = "Test Provider"
  slug = "test-provider"
}

resource "netbox_provider_account" "test" {
  name             = "Test Account"
  account          = "1234567890"
  circuit_provider = netbox_provider.test.name
}
```
