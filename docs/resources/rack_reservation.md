# netbox_rack_reservation (Resource)

Manages a rack reservation in NetBox. Rack reservations allow you to designate specific units within a rack for a particular purpose or user.

## Example Usage

```terraform
resource "netbox_site" "test" {
  name   = "Test Site"
  slug   = "test-site"
  status = "active"
}

resource "netbox_rack" "test" {
  name   = "Test Rack"
  site   = netbox_site.test.id
  status = "active"
  width  = 19
}

data "netbox_user" "admin" {
  username = "admin"
}

resource "netbox_rack_reservation" "test" {
  rack        = netbox_rack.test.name
  units       = [1, 2, 3]
  user        = data.netbox_user.admin.username
  description = "Reserved for testing"
}
```
