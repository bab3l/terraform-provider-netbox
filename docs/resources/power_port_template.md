# netbox_power_port_template (Resource)

Manages a power port template in NetBox. Power port templates define the default power ports that will be created on new devices of a specific device type or modules of a module type.

## Example Usage

```terraform
resource "netbox_manufacturer" "example" {
  name = "Example Manufacturer"
  slug = "example-manufacturer"
}

resource "netbox_device_type" "example" {
  manufacturer = netbox_manufacturer.example.id
  model        = "Example PDU Device Type"
  slug         = "example-pdu-device-type"
}

resource "netbox_power_port_template" "primary_psu" {
  name        = "PSU1"
  device_type = netbox_device_type.example.id
  type        = "iec-60320-c14"
  label       = "Primary PSU Input"
}

resource "netbox_power_port_template" "redundant_psu" {
  name           = "PSU2"
  device_type    = netbox_device_type.example.id
  type           = "iec-60320-c20"
  maximum_draw   = 500
  allocated_draw = 400
  label          = "Redundant PSU Input"
  description    = "Secondary power inlet template for dual-supply devices"
}
```

## Schema

### Required

- `name` (String) The name of the power port template. Use {module} as a substitution for the module bay position when attached to a module type.

### Optional

- `allocated_draw` (Number) Allocated power draw (watts) for this power port.
- `description` (String) Description of the power port template.
- `device_type` (String) The device type ID or slug. Either device_type or module_type must be specified.
- `label` (String) Physical label of the power port template.
- `maximum_draw` (Number) Maximum power draw (watts) for this power port.
- `module_type` (String) The module type ID or model name. Either device_type or module_type must be specified.
- `type` (String) The type of power port (e.g., iec-60320-c6, iec-60320-c8, iec-60320-c14, nema-1-15p, nema-5-15p, nema-5-20p, etc.).

### Read-Only

- `id` (Number) The unique numeric ID of the power port template.

## Import

Import is supported using the following syntax:

```shell
# Power Port Templates can be imported by ID
terraform import netbox_power_port_template.primary_psu 123
```
