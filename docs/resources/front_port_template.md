# netbox_front_port_template (Resource)

Manages a front port template in NetBox. Front port templates define the default front ports that will be created on new devices of a specific device type or modules of a module type. Each front port must map to a rear port.

## Example Usage

```terraform
resource "netbox_manufacturer" "test" {
  name = "Test Manufacturer"
  slug = "test-manufacturer"
}

resource "netbox_device_type" "test" {
  model        = "Test Model"
  slug         = "test-model"
  manufacturer = netbox_manufacturer.test.id
  u_height     = 1
}

resource "netbox_rear_port_template" "test" {
  name        = "Rear Port Template"
  device_type = netbox_device_type.test.id
  type        = "8p8c"
  positions   = 1
}

resource "netbox_front_port_template" "test" {
  name               = "Front Port Template"
  device_type        = netbox_device_type.test.id
  type               = "8p8c"
  rear_port          = netbox_rear_port_template.test.id
  rear_port_position = 1
}
```

## Schema

### Required

- `name` (String) The name of the front port template. Use {module} as a substitution for the module bay position when attached to a module type.
- `rear_port` (String) The name of the rear port template on the same device type or module type that this front port maps to.
- `type` (String) The type of front port (e.g., `8p8c`, `8p6c`, `110-punch`, `bnc`, `f`, `n`, `mrj21`, `fc`, `lc`, `lc-pc`, `lc-upc`, `lc-apc`, `lsh`, `lsh-pc`, `lsh-upc`, `lsh-apc`, `mpo`, `mtrj`, `sc`, `sc-pc`, `sc-upc`, `sc-apc`, `st`, `cs`, `sn`, `sma-905`, `sma-906`, `splice`, `other`).

### Optional

- `color` (String) Color of the front port in hex format (e.g., `aa1409`).
- `description` (String) A description of the front port template.
- `device_type` (String) The device type ID or slug. Either device_type or module_type must be specified.
- `label` (String) Physical label of the front port template.
- `module_type` (String) The module type ID or model name. Either device_type or module_type must be specified.
- `rear_port_position` (Number) Position on the rear port that this front port maps to (1-1024). Default is 1.

### Read-Only

- `id` (Number) The unique numeric ID of the front port template.

## Import

Import is supported using the following syntax:
