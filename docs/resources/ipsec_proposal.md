# netbox_ipsec_proposal (Resource)

Manages an IPSec Proposal in Netbox. IPSec proposals define the security parameters for the IPSec phase 2 (ESP/AH) negotiation in VPN connections.

## Example Usage

```terraform
resource "netbox_ipsec_proposal" "test" {
  name                     = "Test IPSec Proposal"
  encryption_algorithm     = "aes-256-cbc"
  authentication_algorithm = "sha-256"
}
```
