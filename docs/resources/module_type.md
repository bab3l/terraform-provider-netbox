# netbox_module_type (Resource)

Manages a module type in NetBox. Module types define hardware module specifications (model, manufacturer, etc.) that can be instantiated as modules within devices.

## Example Usage

```terraform
resource "netbox_manufacturer" "test" {
  name = "Test Manufacturer"
  slug = "test-manufacturer"
}

resource "netbox_module_type" "test" {
  model        = "Test Module Type"
  manufacturer = netbox_manufacturer.test.id
}
```

- `weight_unit` (String) Unit for weight measurement. Valid values: `kg`, `g`, `lb`, `oz`.
