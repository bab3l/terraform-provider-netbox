# netbox_custom_field_choice_set (Resource)

Manages a custom field choice set in Netbox. Choice sets define the allowed values for selection custom fields.

## Example Usage

```terraform
resource "netbox_custom_field_choice_set" "example" {
  name = "priority_levels"
  extra_choices = [
    { value = "critical", label = "Critical" },
    { value = "high", label = "High" },
    { value = "medium", label = "Medium" },
    { value = "low", label = "Low" },
  ]
}

import {
  to = netbox_custom_field_choice_set.example
  id = "123"
}
```

## Schema

### Required

- `extra_choices` (Attributes List) List of extra choices. Each choice has a value and a label. (see [below for nested schema](#nestedatt--extra_choices))
- `name` (String) Name of the choice set.

### Optional

- `base_choices` (String) Base choice set to inherit from. Valid values: `IATA` (Airport codes), `ISO_3166` (Country codes), `UN_LOCODE` (Location codes).
- `description` (String) Description of the custom field choice set.
- `order_alphabetically` (Boolean) Whether to order choices alphabetically. Defaults to false.

### Read-Only

- `id` (String) Unique identifier (assigned by Netbox).
