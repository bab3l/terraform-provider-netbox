# netbox_virtual_device_context (Resource)

Manages a virtual device context (VDC) in NetBox. Virtual device contexts allow a single physical device to be logically partitioned into multiple virtual devices.

## Example Usage

```terraform
resource "netbox_site" "test" {
  name = "Test Site"
  slug = "test-site"
}

resource "netbox_manufacturer" "test" {
  name = "Test Manufacturer"
  slug = "test-manufacturer"
}

resource "netbox_device_type" "test" {
  model        = "Test Model"
  slug         = "test-model"
  manufacturer = netbox_manufacturer.test.slug
  u_height     = 1
}

resource "netbox_device_role" "test" {
  name    = "Test Role"
  slug    = "test-role"
  color   = "ff0000"
  vm_role = false
}

resource "netbox_device" "test" {
  name        = "Test Device"
  device_type = netbox_device_type.test.model
  role        = netbox_device_role.test.slug
  site        = netbox_site.test.slug
}

resource "netbox_virtual_device_context" "test" {
  name   = "Test VDC"
  device = netbox_device.test.name
  status = "active"
}
```

## Schema

### Required

- `device` (String) The device this VDC belongs to (ID or name).
- `name` (String) The name of the virtual device context.
- `status` (String) Operational status of the VDC. Valid values: `active`, `planned`, `offline`.

### Optional

- `comments` (String) Additional comments or notes about the virtual device context. Supports Markdown formatting.
- `custom_fields` (Attributes Set) Custom fields assigned to this resource. Custom fields must be defined in Netbox before use. (see [below for nested schema](#nestedatt--custom_fields))
- `description` (String) Description of the virtual device context.
- `identifier` (Number) Numeric identifier unique to the parent device.
- `primary_ip4` (String) Primary IPv4 address assigned to this VDC (ID).
- `primary_ip6` (String) Primary IPv6 address assigned to this VDC (ID).
- `tags` (Set of String) Tags assigned to this resource. Tags must already exist in Netbox.
- `tenant` (String) The tenant associated with this VDC (ID or slug).

### Read-Only

- `id` (String) The unique numeric ID of the virtual device context.
