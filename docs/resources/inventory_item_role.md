# netbox_inventory_item_role (Resource)

Manages an inventory item role in NetBox. Inventory item roles define the functional purpose of inventory items within devices.

## Example Usage

```terraform
resource "netbox_inventory_item_role" "test" {
  name  = "Test Role"
  slug  = "test-role"
  color = "ff0000"
}

import {
  to = netbox_inventory_item_role.test
  id = "123"
}
```
