# netbox_circuit_group (Resource)

Manages a circuit group in Netbox. Circuit groups allow you to organize related circuits together for management and reporting purposes.

## Example Usage

```terraform
resource "netbox_circuit_group" "test" {
  name        = "Test Circuit Group"
  slug        = "test-circuit-group"
  description = "Group of related circuits"

  # Partial custom fields management
  # Only specified custom fields are managed, others preserved
  custom_fields = [
    {
      name  = "group_type"
      value = "redundant-pair"
    },
    {
      name  = "cost_center"
      value = "IT-NET-001"
    }
  ]

  tags = [
    "circuit-group"
  ]
}
```
